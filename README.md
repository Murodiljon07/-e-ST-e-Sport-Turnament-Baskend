# 🎮 E-Sport Tournament Platform

Bu loyiha — online e-sport turnirlarni boshqarish tizimi. Admin turnir yaratadi, playerlar ro‘yxatdan o‘tadi, matchlar o‘ynaladi va natijalar asosida reyting avtomatik yangilanadi.

---

## 🚀 Features

- 👤 User Authentication (JWT)
- 🏆 Tournament yaratish va boshqarish
- 🎯 Turnirga join qilish (player/team)
- 📊 Match natijalarini kiritish
- 🔥 Auto ranking system (point asosida)
- ⚡ Real-time updates (Socket.io)
- 🧾 Swagger API documentation

---

## 🛠 Tech Stack

**Backend:**

- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.io

**Frontend (rejalangan):**

- Next.js
- Tailwind CSS

---

## 📂 Project Structure

src/
